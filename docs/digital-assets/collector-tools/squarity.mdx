# Squarity

## Overview

Rarity analyzers are becoming increasingly important in the NFT space. As more and more NFTs are created and sold, it can be difficult for collectors and investors to determine the true rarity and value of individual NFTs. 

This is where [Squarity](https://squarity.app/) comes in!

Squarity provides a one stop shop where you can compare and evaluate the rarity of NFT collections. This can help collectors and investors make more informed decisions about what to do with their NFTs. Watch this quick video to learn more about how to use Squarity!
