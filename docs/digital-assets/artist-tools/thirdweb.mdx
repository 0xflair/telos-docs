# Thirdweb.com

## Overview

Thirdweb is a web3 development framework that offers tools and services for building NFTs, games, and decentralized applications. Its NFT tools include contracts for minting, collection management, and marketplace functionality. 
Thirdweb continuously adds new features and services to support developers in building successful NFT projects.
Key features of Thirdweb's NFT tools include support for the efficient ERC721A standard, lazy minting to save on gas fees, metadata customization, and the ability to set royalties for passive income.
With Thirdweb, developers can create NFT drops, build NFT marketplaces, and design NFT-centric games. It's a user-friendly tool for deploying NFTs and creating engaging experiences.
If you're planning to build an NFT project, Thirdweb is a powerful starting point.
